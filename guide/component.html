# 组件

# 使用第三方组件库

在 .vuepress/enhanceApp.js 中，引入所需组件

// .vuepress/enhanceApp.js
// 全局注册 Element 组件库
import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

export default ({
  Vue,
  options,
  router
}) => {
  Vue.use(Element)
}

注意！！

在引入组件库后，可能会提示类似 Cannot find module 'core-js/library/fn/object/assign 等围绕 core-js 的报错信息，可以尝试以下代码更新或降级 core-js；

npm install core-js@2 / npm install core-js@3
#如果不想更改 core-js 的版本，则可使用：
yarn add async-validator@1.11.5 / npm install async-validator@1.11.5

小提示！

可通过以下代码查询更具体的版本信息

npx vuepress info

# 引入自定义组件

所有在 .vuepress/components 下的 *.vue 文件都会自动地被注册为全局的异步组件。 例如存在 .vuepress/components/my-demo.vue 文件，则在所有的 *.md 文件中可以直接使用：

<my-demo/>
