# 预处理器

VuePress对以下处理器已经内置了相关对webpack配置： sass、scss、less、stylus 和 pug。要使用它们，值需要在项目中安装对应的依赖。例如 sass，需要安装：

yarn add -D sass-loader node-sass / npm install -D sass-loader node-sass

然后就可以在组件或者 Markdown 中使用：

<style lang="sass">
  .title
    background: #87ceeb;
</style>
