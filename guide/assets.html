# 静态资源

# 公共文件

如果有一些静态资源，如用于 favicons 或者 PWA 的图标，可以存放在 .vuepress/public 下，最终他们会被复制到生成的静态文件夹中。

# 基础路径

如果部署的网站是一个非根路径，如 https://bluej.github.io/bar/ ，那需要在 .vuepress/config.js 下设置 base 的值为 "/bar/"。

注意

base 的值需要以 / 开头，并以 / 结尾。

这时如果需要引用到 .vuepress/public 下的静态资源，则需要将路径改写成 /bar/xxx.jpg。但如果某一天需要修改 base，那将会比较麻烦。这时，vuepress 提供了一个内置的 $withBase，具体用法如下：

<img :src="$withBase('/xxx.jpg')">
