# 目录结构

.
├── docs
│   ├── .vuepress
│   │   ├── `components` (可选的)
│   │   ├── `theme` (可选的)
│   │   │   └── Layout.vue
│   │   ├── `public` (可选的)
│   │   ├── `styles` (可选的)
│   │   │   ├── index.styl
│   │   │   └── palette.styl
│   │   ├── `templates` (可选的, 谨慎配置)
│   │   │   ├── dev.html
│   │   │   └── ssr.html
│   │   ├── `config.js` (可选的)
│   │   └── `enhanceApp.js` (可选的)
│   │ 
│   ├── README.md
│   ├── guide
│   │   └── README.md
│   └── config.md
│ 
└── package.json
  • docs/.vuepress: 用于存放全局的配置、组件、静态资源等；
  • docs/.vuepress/public: 静态资源目录（如图片)；
  • docs/.vuepress/theme: 本地主题；
  • docs/.vuepress/config.js: 配置文件的入口文件，也可以是 YML 或 toml。
